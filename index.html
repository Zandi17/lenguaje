Algunos Lenguajes de la web

HTML + CSS + Javacript

  • HTML5

    • HTML5(Hyper Text Markup language, version 5) es la quinta revision importante del lenguaje básico de la World Wide Web, HTML.

    Usa la extension .html

  • CSS 3

    • Hoja de estilo en cascada o CSS(siglas en ingles de cascading stule sheets) es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.

    Usa la extension css

  • JavaScript

    • JavaScript(abreviado comúnmente "JS") es un lenguaje e programacion interpretado utilizado principalmente en el lado del cliente (client-side), implementado parte de un navegador web permitiendo mejorar en la interfaz de usuario y páginas web dinámicas.

    Usa la extension .js
